
# **Custom OAuth Server**

OAuth 인증의 과정 및 로직을 직접 구현해보고 이해하는데 목표가 있습니다.


## **Project Stack**

- JavaScript
- Node.js
- Express.js

## **Run on your machine**

 1. git clone https://github.com/PilYeooong/custom-oauth-server.git
 2. npm i (client / OAuth 폴더 양쪽)
 3. npm start
